First Clone the Project. Then,

From the project root run the following in terminal/command prompt:
- npm install
- npm start


Then follow prompts:
1. Choose product (number or code)
2. Enter quantity
3. Choose action (proceed to payment / add item / cancel)
4. If proceeding, optionally enter promo code
5. See final expected cart items and total

To check scenarios run :
- npm test
